### Option 2: Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/mint-ide.git
   cd mint-ide
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` and start building!
